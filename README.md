# syncer_swim

UTC-based 24/7 playout scheduler for OBS Studio with metadata overlays, hot-reload, and FFprobe metadata extraction.

## Quickstart

See [QUICKSTART.md](QUICKSTART.md).

## Files

- broadcast_scheduler.py: main daemon.
- schedule_manager.py: interactive CLI scheduler.
- obs_metadata_overlay.py: OBS script for metadata overlays.
- setup.sh: one-command setup.
